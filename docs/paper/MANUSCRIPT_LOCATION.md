# Manuscript location

The **living LaTeX manuscript is not tracked in this Git repository.**

| Item | Path |
|------|------|
| Manuscript root | `~/papers/snatch-phase-bench/paper/` |
| Main file | `~/papers/snatch-phase-bench/paper/main.tex` |
| Writing status | `~/papers/snatch-phase-bench/paper/WRITING_STATUS.md` |

Relative path from this repository root (`snatch-phase-bench/`):

```text
../paper/
```

## In-repo paper documentation

These files **stay in Git** and track manuscript *process*, not LaTeX sources:

- [`PAPER_TODO.md`](PAPER_TODO.md) — action tracker
- [`REVIEWER_CHECKLIST.md`](REVIEWER_CHECKLIST.md) — reviewer risks

When experiments complete, update LaTeX tables in the external `paper/` directory, then update `PAPER_TODO.md` and `WRITING_STATUS.md` accordingly.

See [`../SCIENTIFIC_WORKFLOW.md`](../SCIENTIFIC_WORKFLOW.md).
